Data Dictionary



Order Information	Column	Description

Order\_Id	Unique order identifier

Order\_Date	Date when order was placed

Shipping\_Date	Date when order was shipped

Order\_Status	Current order status

Delivery\_Status	Delivery outcome

Shipping\_Mode	Shipping method used



Customer Information	Column	Description

Customer\_Id	Unique customer identifier

Customer\_Fname	Customer first name

Customer\_Lname	Customer last name

Customer\_City	Customer city

Customer\_State	Customer state

Customer\_Country	Customer country

Customer\_Segment	Customer business segment



Product Information	Column	Description

Product\_Card\_Id	Product identifier

Product\_Name	Product name

Product\_Category\_Id	Product category identifier

Category\_Name	Product category

Product\_Price	Product selling price



Sales Metrics	Column	Description

Sales	Revenue generated from the transaction

Sales\_per\_customer	Revenue generated per customer

Benefit\_per\_order	Profit contribution per order

Order\_Profit\_Per\_Order	Net profit generated by order

Order\_Item\_Total	Total item value after adjustments



Discount Metrics	Column	Description

Order\_Item\_Discount	Discount amount applied

Order\_Item\_Discount\_Rate	Discount percentage applied



Logistics Metrics	Column	Description

Days\_for\_shipping\_real	Actual shipping duration

Days\_for\_shipment\_scheduled	Planned shipping duration

Late\_delivery\_risk	Binary indicator for delay risk (1 = Late, 0 = On Time)



Geographic Information	Column	Description

Market	Business market region

Order\_Region	Geographic order region

Order\_Country	Destination country

Order\_City	Destination city

Latitude	Geographic latitude

Longitude	Geographic longitude



Product Performance Metrics	Column	Description

Order\_Item\_Quantity	Quantity ordered

Order\_Item\_Product\_Price	Product price at transaction time

Order\_Item\_Profit\_Ratio	Profit ratio for order item



Machine Learning Target	Column	Description

Late\_delivery\_risk	Target variable used for shipment delay prediction
